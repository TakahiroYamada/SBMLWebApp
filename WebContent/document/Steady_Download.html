Download the result data

After you execute steady state analysis, you can get the result on your computer pressing "Download data". The downloaded data is Zip file to contain table about steady state point and Jacobian matrix. Default name of this file is (File name of SBML model) + "suffix". Of course you can change the name using download dialog.
